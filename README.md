# Consensus Scraper

This Python script extracts text from Consensus PDFs.

## Setup

1. Create a virtual environment:

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

2. Install dependencies:

```bash
pip -r requirements.txt
```

## Functionality

Put pdfs in `/pdfs`. Results will be saved as JSON in `/results`.
